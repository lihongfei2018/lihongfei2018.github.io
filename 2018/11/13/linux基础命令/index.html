显示当前系统使用的SHELL

shell被称为命令解释器,是一种高级语言，他可以把我们输入的命令转换为机器可以识别的语言，（0和1）也可用cat命令查看当前系统装了哪些shell

命令提示符(id -u)(PS1)

“#” 为管理员 “$” 为普通用户
也可输入命令来查看，显示为“0” 就是管理员，不是“0”的皆为普通用户

命令提示符可以通过修改PS1变量来改变字体颜色、亮度等，例：

数字1为显示高亮；数字5为为闪烁；数字41为填充颜色；数字33为字体颜色
\u显示当前用户；\h显示主机名简称；\w显示当前工作目录；\W显示当前工作目录基名

hash 缓存

系统初始hash表为空，当外部命令执行时，默认会从PATH路径下寻找该命令，找到后会将这条命令的路径记录到hash表中，当再次使用该命令时，shell解释器首先会查看hash表，存在的话就执行，如果不存在，将会去PATH路径下寻找。利用hash缓存表可大大提高命令的调用速率。
hash -d +name 清除缓存
hash -r 清除所有的缓存
查看hash表已执行过并保存的命令

在linux中 hash 别名、内部、外部命令的优先级顺序是： 别名——内部命令——hash表——外部命令

screen命令：

screen命令可用在退出远程终端或者断网之后仍然可以继续运行一个耗时很长程序，不会因为退出终端或者断网而停止运行。
输入命令screen 建立，然后运行一个耗时很长的任务。（实验后期补上）

screen也可以远程帮助别人，两人在同一个网络，假如一个人需要帮助，寻求帮助的人可以建立一个screen会话，另一人加入进去。
screen -S +name 创建一个会话（图名字为11）screen -x 11 即可加入会话。
自己临时单方面的退出快捷键ctrl+a，b，exit 关闭 查看已建立的会话 screen -ls

echo命令

echo 功能是回显输入的内容
echo命令的基础用法
echo -e "\a 命令可用使电脑发出声响，比如在运行一个程序的时候时间比较长，可用在命令后面加上，执行完命令之后有提示音可以知道命令执行完毕了，睡眠5秒后发出声响。

echo后面内容加单引号' ' 只会回显输入的内容，不管是内容变量还是命令都不会执行。

echo后面内容加双引号" " 只会回显内容和变量，不能识别内容是不是命令。

echo后面内容加反单引号` `可以识别内容是变量或命令。

history 命令的使用

history可以查看以前输入过的命令历史，并且存在文件夹里.bash_history 可用cat命令查看.

可用history -c清空命令历史，只会清空保存在内存中的历史，不会清空保存在磁盘上的命令历史。

几种常见的用法：

command content
history -a 可以把内存里的命令历史存到硬盘里 -
history -r 可以把保存在硬盘里的命令历史读取到内存里的列表
history -w 可以把历史命令保存到指定文件夹里
history n 只显示指定的几条历史命令
ctrl + r 在命令行历史里搜索命令ctrl+g 退出
esc + . 可以d调用前一个命令的最后一个参数

HISTTIMEFORMAT="%F%T " 可让命令历史显示时间，方便知道什么时候敲过什么命令
