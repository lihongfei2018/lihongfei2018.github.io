标准I/O和管道

标准输入和输出

对于程序来讲在执行命令的时候把数据加以处理然后再把结果显示出来，就需要我们用键盘来输入信息（input）程序处理完之后在输出给我们看（output），错误文件也会输出给我们。
读入数据：Input；输出数据：Output

标准输入（STDIN）－0默认接受来自键盘的输入
标准输出（STDOUT）－1默认输出到终端窗口
标准错误（STDERR）－2默认输出到终端窗口

重定向

重定向是把我们输入的命令的结果输出到别的窗口，默认是自己的终端窗口

input > output 一个大于号如果文件存在那么会覆盖这个文件,不存在则创建
input >> output 两个大于号如果文件存在那么会把要输出的内容追加到文件里,不存在则创建
在 2> 前加数字2错误信息也会保存到文件夹里，文件存在则覆盖，不存在则创建
在2>> 前加数字2错误信息也会保存到文件夹里，文件存在则追加，不存在则创建
执行set -C之后不会覆盖文件，>| 可强行覆盖 set +C开启可以覆盖
如果输出的结果不想别人看到 可以重定向到/dev/null下
&>把所有输出重定向到文件

错误输出重定向

&> 覆盖重定向
&>> 追加重定向
把错误信息放一个文件夹，正确信息放一个文件夹

把所有的输出信息全部放在一个文件夹

2&>1是把错误信息输出为正确信息

tr -t可以把字符转换成指定的字符

tr -d 可以删除指定的字符

tr -c 取反,回车之后ctrl+d 显示结果

tr -s 把相同的压缩
